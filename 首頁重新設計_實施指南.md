## 🐛 常見問題排查

### Q: 動畫沒有顯示？

```
A: 嘗試以下步驟：
1. 刷新頁面（Ctrl+Shift+R 強制刷新）
2. 打開 F12 → Console，查看是否有錯誤
3. 確認瀏覽器支持 CSS Animations
4. 檢查 IntersectionObserver 支持
```

### Q: 按鈕無法點擊？

```
A: 檢查以下項目：
1. 確認滑鼠在按鈕上方
2. 檢查按鈕是否被遮擋
3. 在 Console 中查看 JavaScript 錯誤
4. 嘗試在新標籤頁打開
```

### Q: 手機版顯示不正常？

```
A: 驗證以下內容：
1. 按 Ctrl+Shift+M 切換到行動裝置模式
2. 調整視窗寬度到 <768px
3. 確認所有元素單列顯示
4. 確認按鈕寬度為 100%
```

### Q: 頁面加載很慢？

```
A: 進行以下診斷：
1. 按 F12 → Network 標籤
2. 查看是否有資源載入失敗（紅色）
3. 檢查 CSS 和 JS 是否正確載入
4. 檢查網絡連接速度
```
