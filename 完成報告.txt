【TESES評分系統】 ✅ 完整實現

1. 兩階段評分機制
   ✓ 第一階段：基礎門檻篩選
     - 環保裁罰紀錄查詢
     - 勞工違規紀錄查詢
     - 公司治理評鑑查詢
     - 產業排除清單檢查
   
   ✓ 第二階段：簡易永續績效評分
     - 12個評分指標（可量化、易佐證）
     - 企業自填表單（15-20分鐘）
     - 即時評分計算與等級判定

2. 評分架構 (100分制)
   ✓ E（環境）35分 | 3個指標
     - E1：碳管理意識與盤查（15分）
     - E2：能源效率與節約行動（10分）
     - E3：廢棄物與水資源管理（10分）
   
   ✓ S（社會）30分 | 4個指標
     - S1：員工培訓與職涯發展（10分）
     - S2：員工福利與友善職場（10分）
     - S3：供應鏈管理（初階）（5分）
     - S4：當地社會參與（5分）
   
   ✓ G（治理）25分 | 3個指標
     - G1：永續專責組織與承諾（10分）
     - G2：法規遵循紀錄（10分）
     - G3：誠信經營與風險管理（5分）
   
   ✓ T（轉型透明度）10分 | 2個指標
     - T1：平台使用與透明揭露（5分）
     - T2：永續目標設定（5分）

3. 評分等級與金融優惠
   ✓ A級（80-100分）：領先級
     - 利率減碼 -0.15% ~ -0.2%
     - 產品：SLL、綠色融資
     - 額外福利：額度增加、顧問支援
   
   ✓ B級（60-79分）：平均級
     - 利率減碼 -0.05% ~ -0.1%
     - 產品：永續授信、主題貸款
     - 額外福利：免費顧問諮詢
   
   ✓ C級（30-59分）：潛力級
     - 無利率優惠
     - 產品：一般授信
     - 條件：需在一年內達60分以上
   
   ✓ D級（0-29分）：風險級
     - 利率加碼 +0.05%
     - 產品：一般授信（加嚴審核）
     - 要求：提交改善計畫

🚀 部署指南
===============================================================================

【本地開發環境】

1. 安裝依賴
   $ npm install

2. 啟動伺服器
   $ npm start
   伺服器將運行於 http://localhost:3000

3. 訪問應用
   - 首頁：http://localhost:3000
   - 評分系統：http://localhost:3000/assessment
   - 輔導平台：http://localhost:3000/platform

【生產環境部署】

推薦配置：
- Node.js v18+
- PM2 進程管理
- Nginx 反向代理
- SSL/TLS 加密
- MongoDB 或 PostgreSQL 數據庫
- Redis 緩存層（可選）
- CDN 內容分發（可選）

【測試命令】

評分系統測試
POST http://localhost:3000/api/calculate-score
{
  "e1_carbonManagement": "completed-scope1-2",
  "e2_energyEfficiency": "updated-equipment-past2y",
  ...
}

碳盤查測試
POST http://localhost:3000/api/carbon-calculator
{
  "electricity": 100000,
  "naturalGas": 500,
  ...
}

改善建議測試
POST http://localhost:3000/api/improvement-suggestions
{
  "improvements": ["E1", "S1", "G2"]
}
